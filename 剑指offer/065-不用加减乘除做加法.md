### [65. 不用加减乘除做加法](https://leetcode-cn.com/problems/bu-yong-jia-jian-cheng-chu-zuo-jia-fa-lcof/)

写一个函数，求两个整数之和，要求在函数体内不得使用 “+”、“-”、“*”、“/” 四则运算符号。

 **示例:**

```
输入: a = 1, b = 1
输出: 2
```

**提示：**

- `a`, `b` 均可能是负数或 0
- 结果不会溢出 32 位整数

---

**Java**

``` java
class Solution {
    public int add(int a, int b) {
        // ^ 用于不进位的加法， & 用于算进位
        while(b != 0){
            int x = a ^ b;
            b = (a & b) << 1;
            a = x;
        }
        return a;
    }
}
```

如果说不能用 `“+”、“-”、“*”、“/” `，那么肯定就是位运算

`^`：相同为 0， 不同为 1 

```
two num   add    ^
 0, 1      1     1
 1, 0      1     1
 0, 0      0     0
 1, 1      0     0
```

看上面可以发现仅仅使用 `^`就可以完成二进制的加法，但是唯一需要注意的是进位，也就是最后一行 1,1  的情况，可以用 `&` 模拟进位

`&`：同 1 为 1，其他都为 0

```
two num    &    
 0, 1      0     
 1, 0      0     
 0, 0      0     
 1, 1      1     
```

但是仅仅 a & b 是不够的，因为进位是要 + 到前面的那一位，所以使用 `(a & b) << 1`，但是要注意循环进位的问题，所以外层套个 while 循环，等到不需要进位的时候 （b == 0）的时候就算结束了
