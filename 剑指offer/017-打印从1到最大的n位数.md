### [17. 打印从1到最大的n位数]

输入数字 `n`，按顺序打印出从 1 到最大的 n 位十进制数。比如输入 3，则打印出 1、2、3 一直到最大的 3 位数 999。

**示例 1:**

``` java
输入: n = 1
输出: [1,2,3,4,5,6,7,8,9]
```

说明：

- 用返回一个整数列表来代替打印
- n 为正整数

---

``` java
class Solution {
    public int[] printNumbers(int n) {
        int len = (int) Math.pow(10, n);
        int[] res = new int[len - 1];
        Queue<String> q = new LinkedList<>();
        // 先把 1-9 加入队列
        for(int i = 1; i < 10; i++){
            q.offer(String.valueOf(i));
        }
        int index = 0;
        while(!q.isEmpty()){
            int size = q.size();
            for(int i = 0; i < size; i++){
                String str = q.poll();
                // 不同考虑前面有 0 的情况
                res[index++] = Integer.parseInt(str);
                if(str.length() < n){
                    // 这里可以填充 0
                    for(int j = 0; j < 10; j++){
                        q.offer(str + j);
                    }
                }
            }
        }
        return res;
    }
}
```

这是一个考大数的题，long 也存不下，因此考虑 String，但是 String 去模拟进位的话非常麻烦，因此可以用 String 来模拟**全排列**
