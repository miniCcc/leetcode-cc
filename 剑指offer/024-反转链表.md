### [24. 反转链表](https://leetcode-cn.com/problems/fan-zhuan-lian-biao-lcof/)

定义一个函数，输入一个链表的头节点，反转该链表并输出反转后链表的头节点。

**示例:**

```
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
```

**限制：**

```
0 <= 节点个数 <= 5000
```

---

**Java**

``` java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
// 这是一种比较简单的做法，容易想到，但是效率不高
class Solution {
    public ListNode reverseList(ListNode head) {
        if(head == null) return null;
        // 1. 用栈解决
        Stack<ListNode> stack = new Stack<>();
        while(head != null){
            stack.push(head);
            head = head.next;
        }
        head = stack.pop();
        ListNode res = head;
        while(!stack.isEmpty()){
            ListNode temp = stack.pop();
            head.next = temp;
            head = temp;
        }
        head.next = null;
        return res;
    }
}
```

第二种是原地反转链表：

``` java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public ListNode reverseList(ListNode head) {
        // 特殊情况特判
        if(head == null || head.next == null) return head;
        ListNode x = head;
        ListNode p = x.next;
        ListNode q = p.next;
        x.next = null;
        while(q != null){
            p.next = x;
            x = p;
            p = q;
            q = q.next;
        }
        p.next = x;
        return p;
    }
}
```
