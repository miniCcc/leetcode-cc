### [14- I. 剪绳子](https://leetcode-cn.com/problems/jian-sheng-zi-lcof/)

给你一根长度为`n` 的绳子，请把绳子剪成整数长度的 `m` 段（m、n都是整数，n>1并且m>1），每段绳子的长度记为 `k[0],k[1]...k[m-1]` 。请问 `k[0]*k[1]*...*k[m-1]` 可能的最大乘积是多少？例如，当绳子的长度是8时，我们把它剪成长度分别为2、3、3的三段，此时得到的最大乘积是18。

**示例 1：**

```
输入: 2
输出: 1
解释: 2 = 1 + 1, 1 × 1 = 1
```

**示例 2:**

```
输入: 10
输出: 36
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36
```

**提示：**

- `2 <= n <= 58`

---

**Java**

``` java
class Solution {
    // 动态规划
    public int cuttingRope(int n) {
        int[] dp = new int[n + 1];
        dp[2] = 1;
        for(int i = 3; i < n + 1; i++){
            for(int j = 2; j < i; j++){
                dp[i] = Math.max(dp[i], Math.max(j * (i - j), j * dp[i - j]));
            }
        }
        return dp[n];
    }
}
```

首先明确 `dp[i]` 代表什么，代表的是绳子长度为 `i` 的最大乘积结果，假设我们要剪的绳子长度为 `i`，第一次剪 `j` 长度 ( j >= 2 )，还剩下 `i - j`，所以长度可能是 `j * (i - j)`，或者 `j * (dp[i - j])`

方法二：用的是数学证明，也就是 每一段的长度最好为 3

```java
class Solution {
    public int cuttingRope(int n) {
        if(n == 2) return 1;
        if(n == 3) return 2;
        if(n % 3 == 0) return (int) Math.pow(3, n / 3);
        if(n % 3 == 1) return (int) Math.pow(3, n / 3 - 1) * 4;
        return (int) Math.pow(3, n / 3) * 2;
    }
}
```

根据余数进行区分，因为 10 = 3 + 3 + 4，而 5 = 3 + 2，也就是余数决定了 到底是 * 4 还是 * 2
