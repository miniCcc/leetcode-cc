### [48. 最长不含重复字符的子字符串](https://leetcode-cn.com/problems/zui-chang-bu-han-zhong-fu-zi-fu-de-zi-zi-fu-chuan-lcof/)

请从字符串中找出一个最长的不包含重复字符的子字符串，计算该最长子字符串的长度。

**示例 1:**

```
输入: "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
```

**示例 2:**

```
输入: "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

**示例 3:**

```
输入: "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```

提示：

- `s.length <= 40000`

---

**Java**

``` java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        // 特殊情况特判
        if(s.equals(" ")) return 1;
        Set<Character> set = new HashSet<>();
        char[] strs = s.toCharArray();
        int max = 0;
        int left = 0, right = 0;
        while(right < s.length() && left <= right){
            char c = strs[right];
            if(!set.contains(c)){
                set.add(c);
                right++;
            }else {
                max = Math.max(max, right - left);
                while(strs[left] != c){
                    set.remove(strs[left]);
                    left++;
                }
                left++;
                set.remove(c);
            }
        }
        return Math.max(max, right - left);
    }
}
```

模拟滑动窗口，left 和 right 代表滑动窗口的两边边界，同时用 set 保证不重复，如果发现重复，那么需要移动 left 到重复那个位置的右边（也就是丢弃之前的一部分结果），在移动 left 的过程中还需要维护 set，去掉不要的部分

其实还可以用 字典 做的，因为 `<key, value>`, value 可以直接记录这个字符的位置，那么 left 可以直接移动到相应位置，就不用一步一步的走了，同时 key 是不会重复的，因此还具有 set 去重的功能
