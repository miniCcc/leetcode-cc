### [49. 丑数](https://leetcode-cn.com/problems/chou-shu-lcof/)

我们把只包含质因子 2、3 和 5 的数称作丑数（Ugly Number）。求按从小到大的顺序的第 n 个丑数。

**示例:**

```
输入: n = 10
输出: 12
解释: 1, 2, 3, 4, 5, 6, 8, 9, 10, 12 是前 10 个丑数。
```

**说明:** 

1. `1` 是丑数。
2. `n` **不超过**1690。

---

**Java**

``` java
class Solution {
    public int nthUglyNumber(int n) {
        if(n <= 5) return n;
        Set<Integer> map = new HashSet<>();
        map.add(1);
        map.add(2);
        map.add(3);
        map.add(4);
        map.add(5);
        int count = 5, num = 6;
        while(count < n){
            //if(count == n) break;
            count++;
            if(num % 2 == 0 && map.contains(num / 2)) map.add(num);
            else if(num % 3 == 0 && map.contains(num / 3)) map.add(num);
            else if(num % 5 == 0 && map.contains(num / 5)) map.add(num);
            else count--;
            num++;
        }
        return --num;
    }
}
```

上面这个方法 当 n >= 500 的时候会超时......

``` java
class Solution {
    public int nthUglyNumber(int n) {
        int a = 0, b = 0, c = 0;
        int[] dp = new int[n];
        dp[0] = 1;
        int index = 1;
        while(index < n){
            int mul2 = dp[a] * 2;
            int mul3 = dp[b] * 3;
            int mul5 = dp[c] * 5;
            dp[index] = Math.min(mul2, Math.min(mul3, mul5));
            // 千万不要是 else if,如果有2个相同的值，指针都是需要移动的
            if(mul2 == dp[index]) a++;
            if(mul3 == dp[index]) b++;
            if(mul5 == dp[index]) c++;
            index++;
        }
        return dp[n - 1];
    }
}
```
