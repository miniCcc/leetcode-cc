### [40. 最小的k个数](https://leetcode-cn.com/problems/zui-xiao-de-kge-shu-lcof/)

输入整数数组 `arr` ，找出其中最小的 `k` 个数。例如，输入4、5、1、6、2、7、3、8这8个数字，则最小的4个数字是1、2、3、4。

**示例 1：**

```
输入：arr = [3,2,1], k = 2
输出：[1,2] 或者 [2,1]
```

**示例 2：**

```
输入：arr = [0,1,2,1], k = 1
输出：[0]
```

**限制：**

- `0 <= k <= arr.length <= 10000`
- `0 <= arr[i] <= 10000`

---

**Java**

``` java
class Solution {
    public int[] getLeastNumbers(int[] arr, int k) {
        Queue<Integer> pq = new PriorityQueue<>();
        for(int num : arr){
            pq.offer(num);
        }
        int[] res = new int[k];
        int index = 0;
        while(k > 0){
            res[index++] = pq.poll();
            k--;
        }
        return res;
    }
}
```

使用堆，但是效果不太好，因为每 poll 一个元素，堆就得调整一次，最终 beat 31% 左右

可以使用快排，直接返回前  k 个结果，下面的 beat 72% 左右

``` java
class Solution {
    public int[] getLeastNumbers(int[] arr, int k) {
        Arrays.sort(arr);
        int[] res = new int[k];
        for(int i = 0; i < k; i++){
            res[i] = arr[i];
        }
        return res;
    }
}
```

第三种：计数法 beat 100%

``` java
class Solution {
    public int[] getLeastNumbers(int[] arr, int k) {
        int[] count = new int[10001];
        for(int num : arr){
            count[num]++;
        }
        int[] res = new int[k];
        int index = 0;
        for(int i = 0; i < count.length && index < k; i++){
            //if(count[i] == 0) continue;
            for(int j = 0; j < count[i] && index < k; j++){
                res[index++] = i;
            }
        }
        return res;
    }
}
```
