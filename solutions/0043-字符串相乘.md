### [43. 字符串相乘](https://leetcode-cn.com/problems/multiply-strings/)

给定两个以字符串形式表示的非负整数 `num1` 和 `num2`，返回 `num1` 和 `num2` 的乘积，它们的乘积也表示为字符串形式。

**示例 1:**

```
输入: num1 = "2", num2 = "3"
输出: "6"
```

**示例 2:**

```
输入: num1 = "123", num2 = "456"
输出: "56088"
```

**说明：**

1. `num1 `和 `num2 `的长度小于110。
2. `num1` 和 `num2` 只包含数字 0-9。
3. `num1` 和 `num2` 均不以零开头，除非是数字 0 本身。
4. **不能使用任何标准库的大数类型（比如 BigInteger）或直接将输入转换为整数来处理。**

---

**Java**

``` java
class Solution {
    public String multiply(String num1, String num2) {
        int len1 = num1.length();
        int len2 = num2.length();
        String res = "";
        String longer = num1, shorter = num2;
        if(len1 < len2){
            longer = num2;
            shorter = num1;
        } 
        for(int i = shorter.length() - 1; i >= 0; i--){
            res = add(res, mul(longer, shorter.charAt(i) + "", shorter.length() - i - 1));
        }
        StringBuilder sb = new StringBuilder(res);
        // 去除开头的 0
        for(int i = sb.length() - 1; i >= 0; i--){
            if(sb.charAt(i) == '0') sb.deleteCharAt(i);
            else break;
        }
        if(sb.length() == 0) return "0";
        return sb.reverse().toString();
    }
    // 模拟乘法
    public String mul(String a, String b, int count){
        StringBuilder sb = new StringBuilder();
        // 先补充 0
        for(int i = 0; i < count; i++){
            sb.append("0");
        }
        int y = Integer.valueOf(b);
        int more = 0; // 进位
        for(int i = a.length() - 1; i >= 0; i--){
            int x = Integer.valueOf(a.charAt(i) + "");
            int temp = x * y + more;
            sb.append(temp % 10);
            more = temp / 10;
            
        }
        if(more != 0) sb.append(more);
        return sb.toString();
    }
    // 模拟加法
    public String add(String a, String b){
        StringBuilder sb = new StringBuilder();
        int index = 0, more = 0;
        while(index < a.length() || index < b.length()){
            if(index < a.length() && index < b.length()){
                int x = Integer.valueOf(a.charAt(index) + "");
                int y = Integer.valueOf(b.charAt(index) + "");
                int temp = x + y + more;
                sb.append(temp % 10);
                more = temp / 10;
            }else if(index < a.length()){
                int temp = more + Integer.valueOf(a.charAt(index) + "");
                sb.append(temp % 10);
                more = temp / 10;
            }else if(index < b.length()){
                int temp = more + Integer.valueOf(b.charAt(index) + "");
                sb.append(temp % 10);
                more = temp / 10;
            }
            index++;
        }
        if(more != 0) sb.append(more);
        return sb.toString();
    }

}
```
